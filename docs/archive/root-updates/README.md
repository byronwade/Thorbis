# Root Updates Archive

This directory contains all of the ad-hoc summaries, troubleshooting notes, and implementation reports that previously lived in the repository root.

## Why this folder exists

- Keeps `/` focused on active source code and build configuration
- Preserves the historical write-ups for reference
- Makes it easier to find related updates in one place
