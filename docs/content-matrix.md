# Marketing Navigation Content Matrix

This matrix inventories every unique marketing URL surfaced in the header and footer. Use it to track build status, prioritise copy/SEO, and coordinate design or Supabase content requirements.

Legend:

- **Priority** — H = high, M = medium, L = low.
- **Status** — `Exists` (implemented), `Planned` (in progress/design), `Not Started` (no implementation yet).
